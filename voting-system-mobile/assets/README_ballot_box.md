Place your 3D ballot box image here:

- File path: `assets/ballot_box.png`
- Recommended format: PNG with transparent background (or JPG)
- Recommended size: 800x800 or 400x400 (we use `resizeMode='contain'` so it scales)
- Usage: banner graphic and small icon in Home screen

To add the image, copy your `ballot_box.png` into this `assets/` folder. Then restart Metro bundler:

```bash
npx expo start -c
```
